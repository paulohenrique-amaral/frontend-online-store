<h1 align="left">Projeto Online Store</h1>

###

<p align="left">O projeto Online Store é uma aplicação Front End que oferece todas as principais funcionalidades de um marketplace, utilizando a API do Mercado Livre. Abaixo estão as principais características e funcionalidades do projeto:</p>

###

<h3 align="left">Principais Funcionalidades</h3>

###

<p align="left">✨ Busca de Produtos: Permite aos usuários buscar produtos por categoria ou nome, oferecendo uma ampla variedade de opções.<br>✨ Carrinho de Compras: Adicione produtos ao carrinho e gerencie-os de forma fácil e intuitiva.<br>✨ Simulação de Checkout: Simula um processo de checkout completo, incluindo um formulário de cadastro de cliente, garantindo segurança e uma boa experiência ao usuário.</p>

###

<h3 align="left">Experiência do Usuário</h3>

###

<p align="left">✨ Responsividade: Toda a aplicação é totalmente responsiva, podendo ser acessada de diferentes dispositivos, como desktops, tablets e smartphones.<br>✨ Interface Intuitiva: A interface foi desenhada para ser fácil de usar, proporcionando uma experiência agradável e eficiente.</p>

###

<h3 align="left">Tecnologias Utilizadas</h3>

###

<p align="left">🎲 API do Mercado Livre: Integração completa para buscar produtos.<br>🎲 React com TypeScript: Estrutura principal da aplicação.<br>🎲 Material UI: Componentes de interface de usuário modernos e acessíveis.<br>🎲 Context API: Gerenciamento de estado global.<br>🎲 Custom Hooks: Hooks personalizados para lógica de negócios reutilizável.<br>🎲 Styled-Components: Estilização dos componentes.<br>🎲 Efeitos e Animações: Melhoria da experiência visual do usuário.<br>🎲 ESLint: Garantia das melhores práticas de codificação.</p>

###

<h3 align="left">Instalação</h3>

###

Siga as instruções abaixo para configurar e executar o projeto:

- Clone o repositório em sua máquina local

```bash
  git clone git@github.com:paulohenrique-amaral/frontend-online-store.git
```

- Para instalar as dependências.

```bash
  cd frontend-online-store
  npm install
```

- Para iniciar o servidor de desenvolvimento.

```bash
  npm run dev
```
